# ADR-0005: AGENTS.md is the canonical agent doc; CLAUDE.md removed

- **Status:** Accepted
- **Date:** 2026-07-14
- **Encoded in:** commit `266b862` ("Delete CLAUDE.md")

## Context

The repo carried both `AGENTS.md` (the eve-canonical stub pointing at `node_modules/eve/docs/`) and `CLAUDE.md` (a one-line `@AGENTS.md` reference, evidently auto-generated by tooling). Two files with the same purpose invites drift and confusion about which one tooling reads.

## Decision

Delete `CLAUDE.md`. `AGENTS.md` is the single source of truth for agent-doc conventions in this repo.

## Consequences

- (+) One canonical place for tooling and humans to read.
- (+) New commits that re-introduce `CLAUDE.md` should be flagged and the file re-removed.
- (-) Any tooling that hard-codes a path to `CLAUDE.md` will need updating. (None observed in this repo at the time of this decision.)
